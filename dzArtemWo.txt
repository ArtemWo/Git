



1. Дайте определение ветки (своими словами).

     Ветка помогает видеть нам какие были изменения в комитах, спомощью ветки можно прочесть историю комитов.

2. Укажите команды для создания, переключения, просмотра и удаления веток.

     •команду для создания новой ветки    ----   git branch + название ветки;  
       git checkout -b NAME --- создает и переключает на ветку
      •команду для переключения между ветками    ----   git checkout + название ветки 
        git branch -D название ветки --удаляю ветку локально (на ПК) 
	git branch -d название ветки  --удаляю ветку не локально
	git branch -v  -- список веток
